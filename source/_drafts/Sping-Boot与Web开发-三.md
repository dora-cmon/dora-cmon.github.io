# 简介

使用 SpringBoot 的步骤：

1. 创建 SpringBoot 应用，选中需要的模块；

2. SpringBoot 已经默认将这些场景配置好了，只需要在配置文件中指定少量配置就可以运行起来

3. 自己编写业务代码；

**自动配置原理**:

```
xxxxAutoConfiguration：给容器中自动配置组件；
xxxxProperties:配置类来封装配置文件的内容；
```
