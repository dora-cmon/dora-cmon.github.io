Ubuntu 18.04 使用Komorebi实现动态桌面

发表于2020-04-22|更新于2021-11-24|环境搭建 - Ubuntu
|阅读量:

安装 Komorebi

从 github 项目 release 页面 下载 Komorebi 的 deb 文件，并双击安装即可。

若不成功则使用命令安装：

安装成功后启动软件，若提示缺失文件，使用 apt 安装：

下载动态桌面视频及封面

在 火萤视频桌面-资源分享板块 挑选自己喜欢的动态桌面并下载。

以 Daughter Of The Flame为例，进入页面后下载视频文件，并挑选一张图片保存作为封面。我将两个文件都保存在 ~/Videos/"Daughter Of The Flame" 目录下。

创建动态桌面

搜索 komorebi 打开 Wallpaper Creator：

输入/选择：

  • 名称: Daughter Of The Flame
  • wallpaper 类型: A video
  • 视频位置: path
  • 封面位置: path

选择是否在桌面显示时间及显示位置，我选择不显示，其他选项全部默认：

选择下一步，根据提示在终端中输入命令：

重新启动 Komerebi，右键桌面，选择 Change Wallpaper 即可看到刚才新建的动态桌面，选择它。

关闭 Komorebi 声音

确定后发现动态桌面将视频中的声音播放出来了，但是我不想要音频。经过多方查找，目前没有找到关闭动态桌面的官方方法，也许今后会有更新。

采用迂回战术，打开设置中的 Sound -> Applications，将 Komorebi 静音：

虽然方法不是很优雅，但无奈官方暂无解决办法（也有可能我没有找对），但该方法确实解决问题。

进阶——设置 登录/锁屏 界面

打开设置中的 Background，将 Background 和 Lock Screen 都设置为之前保存的封面。

若要实现密码输入界面雾化效果，请参考文章 Ubuntu 18.04 美化完全攻略 中的 GDM主题设置(登录/锁定界面) 小节。
