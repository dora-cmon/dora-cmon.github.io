еҮҶеӨҮе·ҘдҪңпјҡ

зҺҜеўғпјҡе·ІжңүWin10зі»з»ҹпјҢдёҖеқ—SSDпјҢеҲҶеүІзәҰ128Gе®үиЈ…Ubuntu 18.0зі»з»ҹгҖӮ

е·Ҙе…·пјҡ

  • Ubuntu 18.0 LTS
  • UltraISO

еҲҶеүІзЎ¬зӣҳ

еңЁжӯӨз”өи„‘еҸій”®пјҢжү“ејҖ з®ЎзҗҶ->зЈҒзӣҳз®ЎзҗҶ

зЈҒзӣҳ0жҳҜиҰҒе®үиЈ…еҸҢзі»з»ҹзҡ„зЈҒзӣҳпјҢжҒўеӨҚеҲҶеҢәз”ЁдәҺWin10иҝҳеҺҹпјҢEFIеҲҶеҢәдёәWin10еј•еҜјеҲҶеҢәпјҢеқҮдҝқз•ҷгҖӮ

д»ҺCзӣҳеҺӢзј©128Gз©әй—ҙпјҢеҜ№Cзӣҳ->еҸій”®->еҺӢзј©еҚ·пјҢзӯүеҫ…зі»з»ҹжҹҘиҜўеҸҜеҺӢзј©з©әй—ҙгҖӮиҝҷж—¶пјҢеҫҲжңүеҸҜиғҪеҮәзҺ°еҸҜеҺӢзј©з©әй—ҙиҝңе°ҸдәҺе®һйҷ…з©әй—Із©әй—ҙзҡ„жғ…еҶөпјҢиҝҷжҳҜеӣ дёәж–Үд»¶еӯҳеӮЁе№¶дёҚжҳҜиҝһз»ӯзҡ„пјҢиҖҢеҺӢзј©з©әй—ҙеҝ…йЎ»жҳҜдёҖж®өиҝһз»ӯзҡ„з©әй—ҙгҖӮеҰӮжһңеҮәзҺ°иҝҷз§Қжғ…еҶөпјҢе°қиҜ•д»ҘдёӢи§ЈеҶіж–№жЎҲпјҡ

  • жі•дёҖпјҡCзӣҳеҸій”®->еұһжҖ§->е·Ҙе…·->дјҳеҢ–->йҖүжӢ©Cзӣҳе№¶дјҳеҢ–(иҜҘж–№жі•й’ҲеҜ№жңәжў°зЎ¬зӣҳ)

    иӢҘдёәжңәжў°зЎ¬зӣҳпјҢзЈҒзӣҳж•ҙзҗҶеҗҺеә”еҪ“еҸҜд»ҘеҺӢзј©еҮәжҺҘиҝ‘з©әй—Із©әй—ҙзҡ„еӨ§е°ҸгҖӮ

  • жі•дәҢпјҡиӢҘзЈҒзӣҳдёә SSD жҲ–дёҠиҝ°ж“ҚдҪңж— ж•ҲпјҢеҸҜе°қиҜ•дҪҝз”Ё Disk Genius иҪҜд»¶жү§иЎҢеҺӢзј©еҚ·ж“ҚдҪңпјҢиҝӣе…ҘиҪҜд»¶еҗҺйҖүе®ҡCзӣҳпјҢеҸій”®->и°ғж•ҙеҲҶеҢәеӨ§е°Ҹ->иҫ“е…ҘеҗҲйҖӮзҡ„еӨ§е°Ҹ->ејҖе§Ӣ->и°ғж•ҙжҲҗеҠҹеҗҺдҝқеӯҳжӣҙж”№гҖӮ

    дҪҝз”ЁиҜҘж–№жі•еҸҜиғҪжҸҗзӨәй”ҷиҜҜпјҡ

  • жі•дёүпјҡдҪҝз”Ё Smart Defrag ж•ҙзҗҶз©әй—Із©әй—ҙпјҢйҖүжӢ©Cзӣҳ->жҷәиғҪзЈҒзӣҳж•ҙзҗҶ->еҸҜз”Ёз©әй—ҙзўҺзүҮж•ҙзҗҶгҖӮиҝҷдёҖжӯҘж“ҚдҪңжҜ”иҫғиҖ—ж—¶пјҒ

    дёӢйқўжҳҜж•ҙзҗҶеҗҺзҡ„зЈҒзӣҳж–Үд»¶еҲҶеёғпјҡ

    е®ҢжҲҗзЈҒзӣҳеҸҜз”Ёз©әй—ҙж•ҙзҗҶеҗҺпјҢйҮҚж–°иҝӣе…ҘCзӣҳ->еҸій”®->еҺӢзј©еҚ·пјҢеҸҜеҺӢзј©з©әй—ҙеҹәжң¬жҳҜе…ЁйғЁеҸҜз”Ёз©әй—ҙдәҶгҖӮ

    иҫ“е…ҘеҗҲйҖӮзҡ„еӨ§е°ҸпјҢжү§иЎҢеҺӢзј©еҚ·ж“ҚдҪңпјҡ

    еҸҜе°ҶжңҖеҗҺ829MBзҡ„еҲҶеҢәж јејҸеҢ–еҗҺеҗҲе№¶еҲ°ж–°еҺӢзј©еҮәжқҘзҡ„еҚ·дёӯпјҢдҪҶдёҚжҺЁиҚҗиҝҷй’ҹеҒҡжі•гҖӮиӢҘзі»з»ҹзЈҒзӣҳз®ЎзҗҶж— жі•еҲ йҷӨиҜҘеҲҶеҢәпјҢеҸҜдҪҝз”ЁDisk GeniusеҲ йҷӨпјҢйҖүдёӯиҜҘеҲҶеҢәеҸій”®->еҲ йҷӨеҪ“еүҚеҲҶеҢә->дҝқеӯҳжӣҙж”№пјҢжіЁж„ҸеҲ«еҲ й”ҷеҲҶеҢәгҖӮ

е®үиЈ…Ubuntu 18.0зі»з»ҹ

еҲ¶дҪңеҗҜеҠЁUзӣҳ

дҪҝз”ЁUltraISOжү“ејҖUbuntu 18.0й•ңеғҸж–Үд»¶пјҢеҗҜеҠЁ->еҶҷе…ҘзЎ¬зӣҳжҳ еғҸ->йҖүжӢ©UзӣҳпјҲжіЁж„ҸдёҚиҰҒйҖүй”ҷпјү->еҶҷе…Ҙ(жӯӨдёҫдјҡдёўеӨұUзӣҳдёӯжүҖжңүж–Үд»¶пјҢеҒҡеҘҪеӨҮд»Ҫ)

еҸҜиғҪйңҖиҰҒзҡ„ж“ҚдҪң

е…ій—ӯWin10еҝ«йҖҹеҗҜеҠЁе’ҢSecurity Bootпјҡ

  • жҲ‘е®үиЈ…ж—¶е№¶жңӘе…ій—ӯWin10еҝ«йҖҹеҗҜеҠЁпјҢж•…жӯӨжӯҘеҸҜеҝҪз•ҘпјҢиӢҘеҮәзҺ°й—®йўҳиҖғиҷ‘жҳҜеҗҰиҜҘжӯҘйҖ жҲҗгҖӮ
  • Security BootеҺҹжң¬е°ұжҳҜе…ій—ӯзҡ„пјҲеҸҜеңЁBIOSдёӯе…ій—ӯпјҢж–№жі•иҮӘеҜ»пјүпјҢж•…дёҚзЎ®е®ҡжҳҜеҗҰеҝ…йЎ»гҖӮ

е®үиЈ…Ubuntuзі»з»ҹ

дҝ®ж”№BIOSдёәUEFI ONLY

йҮҚж–°еҗҜеҠЁз”өи„‘пјҢиҝӣе…ҘBIOS->Startup->и®ҫзҪ®UEFI OnlyпјҲдёҚеҗҢз”өи„‘BIOSеҸҜиғҪдёҚеҗҢпјү->жҢүF10дҝқеӯҳйҖҖеҮәпјҡ

**жіЁж„Ҹпјҡ**еҰӮжһңжңүBoot device List F12 OptionйҖүйЎ№еҲҷи®ҫзҪ®дёәEnabledгҖӮ

е®үиЈ…Ubuntu

еңЁи®Ўз®—жңәеҗҜеҠЁиҝҮзЁӢдёӯжҢүF12пјҢйҖүжӢ©д»ҺUзӣҳеҗҜеҠЁпјҲUSB HDDпјүпјҡ

**е»әи®®пјҡ**е®үиЈ…иҝҮзЁӢдёҚиҰҒиҒ”зҪ‘пјҢдёҚиҰҒйҖүе®үиЈ…жӣҙж–°/з¬¬дёүж–№иҪҜд»¶пјҢеӣ дёәеңЁжӣҙжҚўжәҗд№ӢеүҚжӣҙж–°дёӢиҪҪйҖҹеәҰдјҡжһҒж…ўпјҢеҸҜеңЁзі»з»ҹе®үиЈ…еҘҪе№¶жӣҙж”№жәҗеҗҺеҶҚжӣҙж–°гҖӮ

иҝӣе…ҘUзӣҳеҗҺйҖүжӢ©Install UbuntuпјҢеҮәзҺ°йҖүйЎ№еҲҷй»ҳи®Өз»§з»ӯпјҡ

еңЁе®үиЈ…зұ»еһӢж—¶йҖүе®үиЈ…UbuntпјҢдёҺWindows Boot Managerе…ұеӯҳпјҡ

зӮ№еҮ»зҺ°еңЁе®үиЈ…еҗҺпјҢжҸҗзӨәе°ҶеҶҷе…ҘзЈҒзӣҳж”№еҠЁпјҢйҖүжӢ©з»§з»ӯпјҡ

йҖүжӢ©ж—¶еҢәпјҢдёҖиҲ¬й»ҳи®ӨеҚіеҸҜпјҲиӢҘдёҚдёәдёҠжө·/еҢ—дә¬пјҢеҲҷжүӢеҠЁйҖүжӢ©пјүпјҡ

жҺҘзқҖзі»з»ҹе°ұејҖе§ӢиҮӘеҠЁе®үиЈ…пјҢUbuntu 18.0дёҚйңҖиҰҒжүӢеҠЁи®ҫзҪ®еҗ„еҲҶеҢәеӨ§е°ҸпјҢж—§зүҲжң¬еҸҜиғҪйңҖиҰҒгҖӮиӢҘе®үиЈ…ж—§зүҲпјҢеҸҜеҸӮиҖғеҚҡе®ўдәІжөӢ!UEFIеҗҜеҠЁжЁЎејҸдёӢпјҢз”өи„‘е®үиЈ…win10е’ҢUbuntuеҸҢзі»з»ҹиҝӣиЎҢжүӢеҠЁеҲҶеҢәгҖӮ

е®үиЈ…е®ҢжҲҗеҗҺйҮҚеҗҜеҚіеҸҜиҝӣе…ҘUbuntuзҡ„еҸҢзі»з»ҹеј•еҜјз•ҢйқўпјҢз¬¬дёҖдёӘеҚідёәеҗҜеҠЁUbuntuпјҢз¬¬дёүдёӘдёәеҗҜеҠЁWindowsгҖӮ

и®ҫзҪ®й»ҳи®ӨеҗҜеҠЁзі»з»ҹ

еҲҶдёүз§Қжғ…еҶөпјҡв‘ й»ҳи®ӨUbuntuпјӣв‘Ўй»ҳи®ӨWin10пјӣгҖӮ

й»ҳи®Өиҝӣе…ҘUbuntuзі»з»ҹ

иҝӣе…ҘBIOSпјҢи®ҫзҪ®ubuntuдёәз¬¬дёҖйЎ№пјҢWindows Boot Managerдёәз¬¬дәҢйЎ№еҚіеҸҜпјҲеӣҫзүҮйқһиҜҘжғ…еҶөпјүпјҡ

й»ҳи®Өиҝӣе…ҘWin10зі»з»ҹ

иҝҷйҮҢжңүдёӨз§Қи®ҫзҪ®ж–№ејҸпјҲдёӘдәәе–ңж¬ўз¬¬дәҢз§Қпјүпјҡ

  • зӣҙжҺҘиҝӣе…ҘWin10еј•еҜјпјҢиҝӣе…ҘUbuntuйңҖжҢүF12еҶҚйҖүжӢ©зі»з»ҹпјҡ

    иҝӣе…ҘBIOSпјҢдҝ®ж”№еј•еҜјйЎәеәҸпјҢи®ҫзҪ®Windows Boot Managerдёәз¬¬дёҖйЎ№пјҢubuntuдёәз¬¬дәҢйЎ№пјӣ

    з”өи„‘еңЁеҗҜеҠЁж—¶дјҡиҮӘеҠЁиҝӣе…ҘWin10зі»з»ҹпјҢеҰӮжһңжғіиҰҒиҝӣе…ҘUbuntuзі»з»ҹпјҢеҲҷеңЁеҗҜеҠЁж—¶жҢүF12йҖүжӢ©ubuntuзі»з»ҹеҚіеҸҜгҖӮ

  • иҝӣе…ҘUbuntuеј•еҜјпјҢй»ҳи®ӨйҖүжӢ©Windows Boot ManagerпјҢиҝӣе…ҘUbuntuеҸӘйңҖеңЁеј•еҜјз•ҢйқўжҢүдёҠдёӢй”®йҖүжӢ©еҚіеҸҜпјҡ

    иҝӣе…ҘBIOSпјҢи®ҫзҪ®ubuntuдёәз¬¬дёҖйЎ№пјҢWindows Boot Managerдёәз¬¬дәҢйЎ№пјӣ

    иҝӣе…ҘUbuntuзі»з»ҹпјҢзј–иҫ‘/etc/default/grubж–Үд»¶пјҡ

    жӣҙж–°еј•еҜјпјҲupdate-grubе’Ңupdate-grub2е®һйҷ…дёҠе®Ңе…ЁзӣёеҗҢпјүпјҡ
